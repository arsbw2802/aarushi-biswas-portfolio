Projects

Bracelet Aid for the d/Deaf

For our capstone project, we were asked to design and develop a working prototype of a unique and marketable product. The end result was a device which should sit on the wrist of the user, detect crucial sounds in the environment such as the fire alarm, and notify the user in the form of unique visual and haptic feedback. The device’s heart is the ESP32-C3 microcontroller that is responsible for processing and control of the entire system. The sounds are captured and communicated with the microcontroller via the I2S protocol, which can process and run FFTs. C++ libraries were utilized to perform FFTs for accurate sound detection, processing and feedback in under 10 seconds. The project also involved designing a PCB which in addition to the microcontroller, utilized an I2S Mems Digital Microphone, an I2C OLED display, two interrupt driven buttons, and a LRA coin vibration motor. The project received the ‘Runner up for Best Bio-medical device’ award from the ECE department.

Arduino · PCB Design · C++ · Product Development & Testing · User Experience · Leadership

GraderU

The ‘Grader-U’ project was a web app created for my web programming course. The idea came from my own frustrating experience of having to reach out to multiple sources to decide on a course: a single ‘go to’ site would have made this much easier. The response was to design and develop an application which offered students an integrated source of all course information, along with a GPA calculator as a special feature. The project was based on the UIUC course data published by our professor Wade A Fagen-ulmschneider and updated every semester. The data was cleaned and processed to be used for the project. The application was created by utilizing MongoDB, Express.js, REACT and Node.js (MERN). It includes features such as viewing a course’s historic data such as average GPAs, semesters offered and professors who have taught the course. Users can both leave feedback and review other users’ feedback for a particular course or a professor. The special feature of the GPA calculator allows the user to enter all their courses for the semester along with the prediction of where they think they fall on the normal deviation bell curve to find out their predicted GPA for the semester. This project acquired the ‘Most innovative project’ title at the course’s app competition.

JavaScript · MongoDB · Express.js · React · Node.js · HTML · SCSS · Material UI/UX · Product Development · Full-Stack
